깃허브 가입 및 리포 및 브랜치 생성
git init
git add xxx (git add . => add all)
git commit -m "..."
git branch -M 브랜치
처음 접속 :
	github에서 토큰 추가
	git remote add origin https://아이디:토큰@git.hub.com/아이디/리포이름.git
github에서 직접 수정한 경우 : git pull origin 브랜치
로컬에서 강제 push : git push -f origin 브랜치
git push --set-upstream origin main (이후 git push)

git commit -a (add & commit)
